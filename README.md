# Los-Angeles-Violent-Blade-Crimes-2010-2023

This repository contains a data analysis of Los Angeles crime data.
The analysis focuses on violent crimes committed with blades.

Study was conducted using Python 3.13 and the Pandas 3.00 and Plotly 6.00 libraries.

Links to crime datasets:

https://data.lacity.org/Public-Safety/Crime-Data-from-2010-to-2019/63jg-8b9z/about_data

https://data.lacity.org/Public-Safety/Crime-Data-from-2020-to-Present/2nrs-mtv8/about_data
